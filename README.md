# Linkedin Shimmer Effect

Stripe NavBar is a clone of Shimmer Effect from [linkedin](https://www.linkedin.com/)

## Installation

Use the package manager [npm](https://www.npmjs.com/) to [yarn](https://yarnpkg.com/) install.

```bash
yarn install
yarn start
```
